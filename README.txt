WEB CIDADÃO — Pacote de Telas (HTML/CSS/JS)

Como usar
1) Extraia o .zip para uma pasta.
2) Abra o arquivo 'admin.html' no navegador para o painel administrativo.
3) Todas as páginas compartilham o mesmo menu lateral e estilos.
4) Para “Sair”, o link leva para 'login.html' (mock).

Páginas incluídas
- admin.html (Painel Inicial)
- demandas.html, demanda-recebimento.html, demanda-detalhe.html
- projetos.html
- transparencia.html
- indicadores.html
- relatorios.html
- categorias.html
- enquetes.html
- historico.html
- login.html

Observações
- Tudo é estático (mock) em HTML/CSS/JS puro, pronto para integrar com API.
- Arquivo app.js ativa o menu e permite filtro de tabelas.
- Mantive visual limpo, com suas variáveis de cor e radius.
